# UGC Water Bottle Ad Content Pack

## 📌 Project Overview
This project is an AI-generated User Generated Content (UGC) ad content pack for a motivational water bottle. It includes hooks, ad scripts, captions, and prompts created using AI tools.

## 🧃 Product Description
The product is a motivational water bottle designed to help users stay hydrated throughout the day. It includes time markers, a leak-proof design, and is suitable for daily use.

## 🎯 Target Audience
- Students
- Fitness enthusiasts
- Busy professionals

## 📂 Project Contents
- hooks.txt → Attention-grabbing hooks
- scripts.txt → UGC ad scripts
- captions.txt → Social media captions
- prompts.txt → AI prompts used

## 🤖 Tools Used
- ChatGPT (AI content generation)

## 🚀 Outcome
Created a complete UGC ad content pack that can be used for social media marketing across platforms like Instagram, YouTube, and Facebook.
